### What is RainExplorer?
RainExplorer is a browser developed by the same person who made VertexEngine. It's still beta so some features may not be there. But it uses less RAM than Google Chrome.

### What is this?
This is the github repo for the core of RainExplorer. This powers RainExplorer. You can extend and add to this repo. You can also use this Browser to build your own RainExplorer-based Browser.

### Can I use this?
Certainly yes! But there's many caveats to using it as it's just the core, so it lacks privacy features to protect you.
It's also missing many features the original RainExplorer has.
The RainExplorer Core is not meant for casual users, so if you are one, please get out of here until you know what
you are doing.

### Will the main RainExplorer keep it's GNU license?
Yes. RainExplorer will forever be GNU license managed. But it's recommended to use this instead because the actual browser
is very hard to extend, as I did not comment every line. So you will have to hope to know what you are doing.
